# Portfolio - Fabrice Niyonsaba

Full-Stack Developer & graduate in Applied Statistics.

This is a **static portfolio website** built with HTML, CSS, and JavaScript. It is designed to be hosted on **GitHub Pages**.

## Live Site

The portfolio is automatically deployed to GitHub Pages via the GitHub Actions workflow in `.github/workflows/static.yml`.

## Technologies Used

- HTML5
- CSS3 (Responsive Design)
- JavaScript (Vanilla)
- Font Awesome Icons
- Google Fonts (Inter)

## Features

- Responsive navigation with mobile toggle
- Animated skill bars
- Stats counter animation
- Typewriter effect on hero title
- Smooth scrolling
- Back to top button
- Contact form (uses mailto: as a static fallback)
- Profile photo upload (stored in localStorage)

## Local Development

Simply open `index.html` in your browser to view the site locally.

## Deployment

Push to the `main` branch to trigger the GitHub Actions workflow, which deploys the site to GitHub Pages.
